Checkbox Control

This setup is valid because checkbox-like controls are driven by checked, not a scalar text-like value. Angular has a dedicated accessor for native checkboxes for exactly this reason. The mismatch is clearest after checking once and then unchecking: native FormData drops the field, but Angular keeps the last scalar value.
